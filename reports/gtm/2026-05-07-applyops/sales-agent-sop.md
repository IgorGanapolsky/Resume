# ApplyOps Sales Agent SOP

Date: 2026-05-07 Thursday EDT

## Operating Principle

Use public, agent-friendly sales frameworks as checklists, not copied text. Every weekly ApplyOps pass should improve one of three surfaces:

- offer strength
- prospecting output
- message clarity

## Weekly Loop

1. Offer audit
   - Is the pain specific?
   - Is the outcome concrete?
   - Is the price anchored against the cost of wasted applications, lost time, or risky resume claims?
   - Are bonuses useful rather than decorative?
   - Is there a clear reason to buy now without false scarcity?

2. Prospecting plan
   - Build one buyer list per segment:
     - rejected or ghosted job seekers
     - laid-off software engineers
     - career coaches
     - bootcamp communities
     - resume writers who need an audit backend
   - Write 5 message variants per segment.
   - Keep the first message short, specific, and low-pressure.

3. Message clarity
   - Name the customer.
   - Name the problem.
   - Name the plan.
   - Name the risk of doing nothing.
   - Name the successful state.

4. Negotiation support
   - Use calibrated questions in sales replies:
     - What have you already tried?
     - Where do you think the application process is breaking?
     - What would make this worth the fee?
     - How are you deciding whether to keep applying the same way?

5. Trust safeguards
   - Never promise interviews or offers.
   - Never sell fabricated credentials or deceptive resume edits.
   - Never pitch blind mass applying.
   - Always preserve user approval before final submit.

## Daily Output

Each daily revenue run should produce:

- 1 offer improvement
- 3 short social posts
- 5 outbound messages
- 1 follow-up message
- 1 objection-handling note

## Core Hooks

- Your AI resume may be lying about you.
- Applying more will not fix a bad application system.
- The first job-search fix is not volume. It is trust.
- A resume that sounds impressive but is not defensible is a liability.
- Stop using AI to spray applications. Use it to build a cleaner pipeline.

## Objection Responses

Too expensive:

The fee is for the diagnostic and operating plan, not a magic guarantee. If the current process is wasting weeks and producing rejections, the first win is finding what is broken and stopping risky claims before they create more damage.

Can AI do this for free?

AI can draft, but it can also hallucinate credentials, metrics, and achievements. ApplyOps is built around truth-locking, evidence, and human approval.

Can you apply for me?

We can prepare and support applications, but we do not blind-submit or bypass job-board rules. Final submits stay user-approved.

Will this get me interviews?

No one can honestly guarantee interviews. The promise is a cleaner, more targeted, more defensible application system.
